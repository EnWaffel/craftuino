---- WHAT IS craftuino? ----
craftuino is a compiler / transpiler that converts Minecraft-like datapack commands into C++ code for Arduino.
craftuino is also capable of uploading to any microcontroller that Arduino supports.

(currently needs the ArduinoIDE to be installed on the user's system)

----- INSTALLATION -----
Run the setup script for your operating system. (and installArduinoCLI.bat for Windows) After the script finished run the build script for your operating system and you're good to go!

craftuino currently support Windows and Linux.

----- USAGE -----
CLI: craftuino -s "my_sketch_directory"
Don't forget to specify the USB port your Arduino is connected to in the config!
